# Hero - PYE 2026

## Componente

Hero principal de la página de inicio.

---

## Objetivo

Presentar inmediatamente la propuesta de valor de Poder y Energía, transmitiendo confianza, profesionalismo y especialización en soluciones energéticas.

El Hero debe responder en los primeros segundos a tres preguntas del visitante:

- ¿Qué hace la empresa?
- ¿Qué soluciones ofrece?
- ¿Qué acción debe realizar el usuario?

## Imagen

La imagen principal debe ubicarse en:

```
assets/img/hero/
```

Nombre recomendado:

```
hero-home.webp
```

Las imágenes utilizadas deberán ser generadas específicamente para Poder y Energía manteniendo una identidad visual uniforme.

---

## Contenido

El Hero contiene:

- Eyebrow
- Título principal (H1)
- Descripción
- CTA principal
- CTA secundario
- Indicador de desplazamiento

---

## SEO

Debe existir únicamente un H1.

Las palabras clave principales son:

- UPS
- Energía Solar
- Infraestructura Eléctrica
- Soluciones Energéticas

---

## Responsive

Desktop

- Contenido alineado a la izquierda.

Tablet

- Ajuste del fondo.
- Conserva la composición.

Mobile

- Contenido centrado.
- Botones a ancho completo.
- Se oculta el indicador de scroll.

---

## Accesibilidad

- HTML semántico.
- Botones accesibles.
- Alto contraste.
- El contenido nunca debe formar parte de la imagen.

---

## JavaScript

Este componente no requiere JavaScript.

Toda la interacción se resuelve mediante HTML y CSS.

Si en el futuro se incorpora alguna funcionalidad dinámica, podrá añadirse un archivo `script.js` sin modificar la estructura del componente.

---

## Convenciones

- No incrustar texto dentro de la imagen.
- Mantener el Hero ligero y rápido.
- No utilizar carruseles.
- No utilizar sliders automáticos.
- Priorizar el rendimiento y la accesibilidad.
